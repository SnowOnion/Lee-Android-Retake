# Usage

It plays some music.

1. Select a music
   ![](doc/Screenshot_1490348139.png)

2. 切歌, then it plays your selected music
   ![](doc/Screenshot_1490348166.png)

# Requirement

It fetches music file from [qiniu.com](qiniu.com), a file server, so it needs network connection.
